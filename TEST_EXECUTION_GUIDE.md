# 功能测试执行指南

> 版本: v1.2 | 基于 REQUIREMENTS_AND_TEST.md 第8节

## 前置条件

| 项目 | 要求 |
|------|------|
| Android 设备 | API 24+ (推荐 API 30+)，已开启 USB 调试 |
| 安装 APK | `.\gradlew.bat assembleDebug && adb install -r app\build\outputs\apk\debug\app-debug.apk` |
| Google Maps API Key | 已配置在 `local.properties` |
| 摄像头权限 | 已授权 |
| 位置权限 | 已授权（用于 GPS 测试） |
| TFLite 模型 | `app/src/main/assets/detect.tflite` 存在 |
| 测试物体 | 准备花瓶、瓶子、剪刀等 COCO 标签对应物品 |

---

## 测试执行命令

```bash
# 安装 APK
.\gradlew.bat assembleDebug
adb install -r app\build\outputs\apk\debug\app-debug.apk

# 查看日志
adb logcat -c && adb logcat -s MuseumGuide CameraFragment ObjectDetector

# 运行单元测试
.\gradlew.bat app:test

# 查看测试报告
start app\build\reports\tests\testDebugUnitTest\index.html
```
